# Warehouse and SQL Surface

The scaffold exports a local SQLite warehouse by default:

`data/warehouse/portfolio_platform.db`

## Exported Objects

- `portfolio_projects`
- `repository_registry`
- `v_portfolio_scores`

## SQL Assets

- `sql/ddl/`
- `sql/analytics/`

The repository also includes a dbt-ready starter structure under `dbt/` aligned to the SQLite export path.
